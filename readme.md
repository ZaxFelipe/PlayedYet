## Minha versão do Backlogged / zerei.gg

### Objetivo
Inspirado na planilha para catalogar jogos do youtuber Cogumelando e Backloggd
criei minha própria versão de anotar meu tempo de jogo ao longo do tempo

### Funcionalidades
- A Principal funcionalidade é adicionar jogos com todas as informações que julgo importantes
como tempo de jogo, data de finalização e plataforma podendo pesquisar e filtrar por elas

- Pagina de estatisticas mostrando tempo total jogado por Tag, Genero

- Download automatico de capas usando rawg (coloque sua api key na pagina de configurações)
